# Foxnic-Samples

![Foxnic](foxnic-web/web-full/view/view-example/src/main/resources/static/images/foxnic-250.jpg)


## 项目介绍
用于展示和提供 Foxnic 相关的示例工程和模版代码

## 目录结构


```
├─common // 通用目录
  │  │─domain // 实体、常量、枚举、实体关系等
  │  │─framework // 项目的公共代码
  │  │─generator // 代码生成
  │  └─proxy // 接口路径定义、Feign客户端代码等
  ├─service  // 后端服务项目
  │  │─service-a // 接口服务项目A
  │  └─service-b // 接口服务项目B
  ├─view  // 前端项目
  │  │─view-a // 前端项目A
  │  └─view-b // 前端项目B
  └─wrapper // 项目聚合
     │─wrapper-a // 聚合项目A
     └─wrapper-b // 聚合项目B
```
